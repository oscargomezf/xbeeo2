Requirements:

- Folder hw. The schematics of hardware used.

- Folder script. The scripts used in this project.

- Install python library Xbee 2.1.0 https://pypi.python.org/pypi/XBee

	$ wget https://pypi.python.org/packages/source/X/XBee/XBee-2.1.0.tar.gz
	$ tar -xzvf XBee-2.1.0.tar.gz
	$ cd XBee-2.1.0
	# python setup.py install

- Install python library Serial: 

	# apt-get install python-serial
	
- Install wiringPi2 library if you want to use two leds to indicate which relay is activated.
